// 01 — Tentang

Tentang Saya

Saya Andri, mahasiswa Informatika di Tangerang Selatan yang sedang menekuni Back-End development. Ketertarikan saya bermula dari rasa ingin tahu: bagaimana sebuah halaman web bisa terlihat indah sekaligus terasa nyaman saat digunakan.

Saat ini saya fokus memperdalam HTML, CSS, dan JavaScript modern, sambil bereksperimen dengan Android Stuido dan Unity untuk proyek-proyek pribadi. Bagi saya, detail kecil — transisi, jarak antar elemen, pemilihan tipografi — adalah yang membedakan produk biasa dari produk yang terasa matang.

Di luar coding, saya suka mengeksplorasi desain antarmuka futuristik dan membaca dokumentasi teknologi baru.
